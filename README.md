# avatarbox.app

app server for [avatarbox.io](https://avatarbox.io)

---

## Getting Started

**`avatarbox.app`** is a [Next.js](https://nextjs.org) application that allows users to set up automated Gravatar updates.

## Installation

```sh
$ npm install
```

## Launching the app

```bash
$ npm run start
```

## Tests

```bash
# unit tests
$ npm run test

# end-to-end tests
$ npm run test:e2e

# test coverage
$ npm run test:cov

# acceptance tests
$ npm run test:spec
```

## Usage

Open http://localhost:8080 to view it in the browser.
